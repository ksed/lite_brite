Lite Brite Simulator

Basic Instructions:

Basic coloring: Click in any block of the Lite-Brite canvas to apply/unapply the current paint.
Other colors: You can cycle through the color list with your up/down keys.
Add colors: Enter new colors to the list using hex codes.
Canvas size: Input number of canvas rows and columns, and click the "Update" button.
